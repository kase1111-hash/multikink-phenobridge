# multikink-phenobridge
Experimental validation bridge for the Multi-Kink Generation Framework (Paper V). Maps KK flavor predictions to CMS Open Data and HEPData measurements. Includes analysis of 61.5M dimuon events, P'5 angular observables, contact interaction Λ limits, and C9-C'9 Wilson coefficient plane. Reproducible pipeline using uproot, no CMSSW required.
